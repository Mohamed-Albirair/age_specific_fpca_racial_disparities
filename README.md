# Age-Specific fPCa Racial Disparities

## Repository Overview

This repository contains the reproduction code for the manuscript:
"Age-specific racial disparities in the incidence of fatal prostate cancer: an analytic deconstruction"
by Albirair M, et al. JNCI Cancer Spectrum (2025). https://doi.org/10.1093/jncics/pkaf103

## Reproducibility

All figures and tables from the published analysis can be regenerated using the R code in this repository, in conjunction with publicly available data from the [SEER*stat](https://seer.cancer.gov/seerstat/) program. The data were sourced using extraction instructions provided in the `R/inputs` folder.

## Folder structure

* `/scripts`: contains the script for the analysis.

* `/functions`: contains the custom functions designed for and used in the analysis.

* `/inputs`: contains instructions for sourcing analysis inputs for analysis from [SEER*stat](https://seer.cancer.gov/seerstat/).

* `/outputs`: contains outputs created by the scripts.
